105. 从前序与中序遍历序列构造二叉树

Construct Binary Tree From Preorder and Inorder Traversal

Posted by 松溪 on Monday, January 9, 2023

可以使用递归的方法来构造二叉树。假设给定的前序遍历序列为 preorder 和中序遍历序列为 inorder。

首先，可以通过前序遍历序列的第一个元素来找到根节点。然后，在中序遍历序列中找到根节点的位置，这样就可以将中序遍历序列划分为左子树和右子树。在划分后的左子树和右子树中，可以通过前序遍历序列中的元素来构造左子树和右子树。

代码实现如下：

func buildTree(preorder []int, inorder []int) *TreeNode {
    if len(preorder) == 0 || len(inorder) == 0 {
        return nil
    }

    root := &TreeNode{Val: preorder[0]}
    for pos, node := range inorder {
        if node == root.Val {
            root.Left = buildTree(preorder[1:pos+1], inorder[:pos])
            root.Right = buildTree(preorder[pos+1:], inorder[pos+1:])
            break
        }
    }
    return root
}
