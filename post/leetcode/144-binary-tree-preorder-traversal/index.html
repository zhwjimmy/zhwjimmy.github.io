144. 二叉树的前序遍历

Binary Tree Preorder Traversal

Posted by 松溪 on Tuesday, January 10, 2023

遍历二叉树指的是在二叉树的每个节点上都执行某个操作。二叉树遍历有三种常用的方法：前序遍历、中序遍历和后序遍历。

前序遍历： 在前序遍历中，我们先访问根节点，然后递归地遍历左子树，再递归地遍历右子树。

中序遍历： 在中序遍历中，我们先递归地遍历左子树，然后访问根节点，再递归地遍历右子树。

后序遍历： 在后序遍历中，我们先递归地遍历左子树，再递归地遍历右子树，然后访问根节点。

二叉树的遍历可以用递归的方式来实现，也可以用非递归的方式来实现，如果使用非递归的方式实现，可以使用栈来帮助遍历。

本题代码实现如下：

func preorderTraversal(root *TreeNode) []int {
    var result []int

    if root != nil {
        result = append(result, root.Val)
        result = append(result, preorderTraversal(root.Left)...)
        result = append(result, preorderTraversal(root.Right)...)
    }

    return result
}
