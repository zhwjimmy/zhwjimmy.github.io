206. 反转链表

Reverse Linked List

Posted by 松溪 on Friday, January 13, 2023

反转链表算法题可以使用迭代或递归来实现。

迭代法

迭代解法中，我们使用三个指针，一个指向当前节点，一个指向当前节点的前一个节点，一个指向当前节点的后一个节点。我们不断地将当前节点的next指针指向前一个节点，然后将前一个指针和当前指针向后移动，最后返回新的链表头。

代码实现如下：

/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func reverseList(head *ListNode) *ListNode {
    var prev *ListNode
    current := head
    for current != nil {
        next := current.Next
        current.Next = prev
        prev = current
        current = next
    }
    return prev
}

本题中，使用三个指针来反转链表：

  • prev: 指向当前节点的前一个节点
  • current: 指向当前节点
  • next: 指向当前节点的下一个节点

首先将 prev 初始化为 nil， current 初始化为链表的头节点。

然后进入循环，对于每一个当前节点，首先将 next 指针指向当前节点的下一个节点。这样做的目的是为了保证在反转当前节点的next指针后，仍然能够继续遍历链表。

接着，将当前节点的 next 指针指向 prev，这样就完成了当前节点的反转操作。

最后，将 prev 赋值为当前节点，current 赋值为 next，这样就能够继续遍历链表，并将下一个节点进行反转。

当遍历完整个链表后，prev指向的就是反转后的链表的头节点，返回prev即可

这里需要注意的是，链表为空或只有一个节点时，不需要执行反转操作，直接返回链表的头节点即可。
